🚀 OpenAI to NVIDIA NIM Proxy

OpenAI compatible proxy for NVIDIA NIM API

Status: ✓ Running
Service: OpenAI Proxy
Available Endpoints
GET/health

Check the health status of the proxy and view configuration.

GET/v1/models

List all available OpenAI-compatible models.

POST/v1/chat/completions

Send chat completion requests (OpenAI compatible format).

Configuration

Required environment variables:

NIM_API_KEY

Your NVIDIA NIM API key

Optional environment variables:

NIM_API_BASE

NVIDIA NIM API base URL (default: https://integrate.api.nvidia.com/v1)

SHOW_REASONING

Set to 'true' to display model reasoning with <think> tags

ENABLE_THINKING_MODE

Set to 'true' to enable thinking mode for supported models

⚠️ Important: Make sure NIM_API_KEY is set in your environment before using the proxy.
